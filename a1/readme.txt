The purpose of this assignment is to plan and create a python script that will allow date strings given in the following formats to be returned in standard format of mmm d, yyyy.
Following are accepted date string formats:
YYYYMMDD
YYYY/MM/DD
YYYY-MM-DD
YYYY.MM.DD

If any errors occur ex. if the wrong date format is given, then an error code will be written.

Example input/output

python3 a1_vktran2.py 2020-10-10
Oct 10, 2020

python3 a1_vktran2.py 202-10-10
Error 09: wrong date entered
